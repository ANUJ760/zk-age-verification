# zk-age-verifier

Zero-knowledge age verification with Circom + snarkjs + Solidity.

The circuit proves a user is 18+ without revealing their birth year.  
The `AgeGate` contract accepts a Groth16 proof and marks the caller as verified.

## Stack

- Circom 2
- snarkjs
- Hardhat 2 + ethers + chai
- Solidity 0.8.28

## Project layout

- `circuits/ageCheck.circom`: age >= 18 circuit
- `scripts/build_circuit.sh`: compile circuit, run trusted setup, export verifier
- `contracts/Verifier.sol`: autogenerated Groth16 verifier
- `contracts/ageGate.sol`: app contract that gates access with zk proof
- `test/ageGate.test.ts`: Hardhat test that generates proof with snarkjs
- `frontend/index.html`: minimal browser demo

## Prerequisites

- Node.js 18+ and npm
- `circom` installed and in PATH
- `snarkjs` available (local npm install is enough; global optional)

## Install

```bash
npm install
```

## Build circuit + keys + verifier

```bash
chmod +x scripts/build_circuit.sh
./scripts/build_circuit.sh
```

This generates:

- `build/ageCheck.r1cs`
- `build/ageCheck_js/ageCheck.wasm`
- `zkeys/ageCheck_0000.zkey`
- `zkeys/ageCheck_final.zkey`
- `zkeys/verification_key.json`
- `contracts/Verifier.sol`

## Run tests

Run only the age gate test:

```bash
npx hardhat test test/ageGate.test.ts
```

The test expects:

- `build/ageCheck_js/ageCheck.wasm`
- `zkeys/ageCheck_final.zkey`

If those files are missing, run `./scripts/build_circuit.sh` first.

## Deploy

Deploy with Hardhat script/console (this repo no longer includes the template `Counter` Ignition module).

Example env vars for Sepolia:

```bash
export SEPOLIA_RPC_URL="https://..."
export SEPOLIA_PRIVATE_KEY="0x..."
```

## Frontend demo

Serve `frontend/` with a local server (do not open directly with `file://`):

```bash
npx serve frontend
```

Then open the served URL and set your deployed `AgeGate` address in `frontend/index.html`.

## Common issues

- `HH411 forge-std not installed`  
  Cause: Hardhat is trying to compile a Foundry test in `contracts/`.  
  Fix: keep Foundry tests outside `contracts/`.

- `HH502 Couldn't download compiler version list`  
  Cause: no network / blocked compiler download.  
  Fix: run with internet access or configure local compiler caching.

- TypeScript red squiggles on `snarkjs` import  
  This repo includes `types/snarkjs.d.ts` for editor compatibility.
